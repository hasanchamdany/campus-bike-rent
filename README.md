# campus-bike-rent
Tugas Pengembangan Aplikasi web

# Package included:
- install express
```
npm add express
```
- install nodemon
```
npm add nodemon
```
- install mongosh
```
npm add mongosh
```
- install mongoose
```
npm add mongoose
```
- install dotenv
```
npm add dotenv
```

# Setting .env file:
- Buat file dengan nama ".env" dalam folder campus-bike-rent di local repository
- Isi file:
<br>PORT: 3000 
<br>DB_CONNECTON: mongodb+srv://paw5:paw5@cluster0.1mmny7r.mongodb.net/?retryWrites=true&w=majority

# Run
```
npm run dev
```
